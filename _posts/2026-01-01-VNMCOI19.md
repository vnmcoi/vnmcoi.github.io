`mod` hay `modulo` là phép toán đồng dư tức là lấy phần dư của phép chia.

Và trong C++ thì phép `mod` được thực hiện bằng toán tử `%`.

Ví dụ về phép chia lấy dư:

Ta có: `A = 5` và `B = 3`. Thường thì chúng ta thường hay gọi là `5` chia `3` bằng `1` dư `2` ở đây `2` chính là `mod`.

Thì kết quả của phép `mod` là một số nguyên dương `r` thỏa mãn `B * k + r = A`.

Trong đó: `k` là số nguyên dương lớn nhất sao cho `B * k <= A`.

Vậy ta có `3 * 1 + 2 = 5` vậy kết quả của phép `mod` là `2`.

Còn để tìm nhanh `k` thì trong C++ chúng ta đã biết phép chia số nguyên có thể tự triệt tiêu đi phần số thực và chỉ giữ lại phần nguyên.

Vậy ta có code để tìm `mod` ví dụ bên dưới:

```

#include <bits/stdc++.h>
using namespace std;

int A, B;

int main()
{
    ios::sync_with_stdio(0);
    cin.tie(0);

    cin >> A >> B;
    int k = A / B;
    int r = A - B * k;
    cout << r;
    return 0;
}

```

**Mở rộng:** Vì sao cần phải có `mod`?

Trong máy tính không có gì là vô tận cả số nguyên, số thực, ... Mọi thứ đều có giới hạn của nó nên người ta sử dụng `mod` để miêu tả một con số nếu nó quá lớn.
