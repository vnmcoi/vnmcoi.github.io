Tiếp theo, chúng ta sẽ đến với một bài toán đầu tiên của phần điều kiện.

Trong C++ khi chúng ta muốn với mỗi một đầu vào một điều kiện gì đó chúng ta muốn thao tác khác nhau thì chúng ta có các hàm `if, else if, else` để hỗ trợ thực hiện.

**Lưu ý:** Việc nếu điều kiện `if` đã được kích hoạt thì các điều kiện `else if, else` bên dưới sẽ không bị kích hoạt ngược lại nếu chỉ là `if` thì điều kiện sẽ được kích hoạt riêng lẻ.

Ví dụ:

```

#include <bits/stdc++.h>
using namespace std;

int A;

int main()
{
    ios::sync_with_stdio(0);
    cin.tie(0);

    cin >> A;
    if (A < 10) {
        cout << "A < 10";
    }
    else {
        cout << "A < 20";
    }
    return 0;
}

```

Và

```

#include <bits/stdc++.h>
using namespace std;

int A;

int main()
{
    ios::sync_with_stdio(0);
    cin.tie(0);

    cin >> A;
    if (A < 10) {
        cout << "A < 10";
    }
    if (A < 20) {
        cout << "A >= 10";
    }
    return 0;
}

```

Với code thứ hai nếu điều kiện $A < 10$ kích hoạt thì cũng sẽ kích hoạt $A < 20$ trong một số trường hợp chúng ta sẽ không biết khi nào điều kiện có trùng hay không nên hãy sử dụng luân phiên `if, else if, else` thay vì chỉ `if`.
