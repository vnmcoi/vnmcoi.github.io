Với những yêu cầu lặp lại nhiều lần thì ta có thể sử dụng vòng lặp `for` (xác định được mỗi bước lặp) và `while` (không xác định được mỗi bước lặp).

Với code mẫu thì chúng ta sử dụng `for` nhưng cũng sẽ có một đoạn code demo cho `while`.

```

#include <bits/stdc++.h>
using namespace std;

int L, R;

int main()
{
    ios::sync_with_stdio(0);
    cin.tie(0);

    cin >> L >> R;
    int i = L;
    while (i <= R)
    {
        cout << i << ' ';
        ++i;
    }
    return 0;
}

```
