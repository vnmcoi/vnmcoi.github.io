Với bài này chúng ta có thể sử dụng `for` hoặc `while` tùy ý.

Code mẫu cho `while`:

```

#include <bits/stdc++.h>
using namespace std;

int N;

int main()
{
    ios::sync_with_stdio(0);
    cin.tie(0);

    cin >> N;
    long long ans = 1;
    int i = 1;
    while (i <= N)
    {
        ans *= i;
        ++i;
    }
    cout << ans;
    return 0;
}

```
