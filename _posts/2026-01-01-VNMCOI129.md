An undirected graph is a tree if and only if it satisfies both of the following conditions:

1. The graph has exactly $N - 1$ edges in other word $M = N - 1$.
2. The graph is **connected** (all vertices belong to a single connected component).

**Implementation:**

We can easily check the first condition in $\mathcal{O}(1)$ time. For the second condition, we can use a simple Graph Traversal algorithm.
