Với bài này chúng ta có thể sử dụng `for` hoặc `while` tùy ý.

Code mẫu cho `while`:

```

#include <bits/stdc++.h>
using namespace std;

int N;

int main()
{
    ios::sync_with_stdio(0);
    cin.tie(0);

    cin >> N;
    int i = (N / 2) * 2;
    while (i >= 2)
    {
        cout << i << ' ';
        i -= 2;
    }
    return 0;
}

```

$N / 2 \times 2$ ở đây là đổi số chẵn / lẻ về số chẵn gần nhất (làm tròn xuống) vì trong C++ thì khi ta chia cho $2$ thì chương trình tự động bỏ phần dư khi đó ta nhân lên thì ta được số chẵn tương ứng.
